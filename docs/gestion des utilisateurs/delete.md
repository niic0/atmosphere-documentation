# Suppression d'un utilisateur

!!! warning "admin only !"

    Cette documentation est destinée uniquement aux administrateurs du site seulement.

## Accéder à la page de suppression d’un utilisateur

La suppression d'un utilisateur se fait depuis la [page d'administration d'atmosphère](https://atmosphere-chartres.fr/admin/). Une fois sur cette page, faites défiler jusqu'à la section **"Utilisateurs"**, puis cliquez sur le profil que vous souhaitez supprimer.

![users](../images/users.png)

Vous accéderez à un formulaire déjà rempli avec les informations actuelles de l'utilisateur.

## Supprimer un utilisateur

Pour supprimer un utilisateur, cliquez simplement sur le bouton **"Supprimer l'utilisateur"**.

![users](../images/delete_user.png)

Une fois la suppression faites, vous obtiendrez le message suivant:
